`threat-tide` is a personal vulnerability/security aggregator feed linking to `piratemoo.com/vulns/`. It’s built to quickly skim through newly disclosed CVEs, pocs, writeups, and useful research without noise.
- Recent CVEs
- Public PoC references
- Researcher attribution
- Simple feed-style

The project is meant to be fast and easy to browse. Just open the feed, skim what’s new, and decide what’s worth investigating.
